Installation Procedure

  • FOR LOCAL INSTALLATION OR SETUP
    1. STEP ONE:
    2. Install Wamp server or Xamp .... But we will recommend either wamp or xamp
    3. Copy and paste the zip folder you donwloaded from our site to you server directory
    4. Extract the zip file.
    5. Create a new database from your server mysql(phpmyadmin).
    6. NOTE: Do not import or copy and paste the install.sql file, the software installation form will do that for you.
    7. go to your browser to lunch the software.
    8. STEP 2:
    9. supply installation code. This can be found in the software folder(installation code.txt) file supply you administrator login detail. For example:
    10. Email : admin@admin.com
    11. Password : admin
    12. Confirm Password admin
    13. and click on next button below the screen
    14. STEP 3:
    15. Supply your hosting detail
    16. database details. For example:
    17. Server: localhost
    18. Username: the username you created
    19. Database: the database you have created
    20. Password: the password you have created for the database
    21. click on finish
    22. click on done!
    23. Automatically the software upload the install.sql to the database
